# Sobre esta pasta
Esta pasta contém dados extraídos da base de dados do IBGE de 2010, prontos para uso com MySQL

## Comandos para reduzir a tabela lista_localidades para cidades

```sql
INSERT INTO cidades
SELECT * FROM lista_localidade WHERE CD_NIVEL = 1
```
